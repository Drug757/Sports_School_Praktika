# Система работы спортивной секции

Веб-приложение для управления спортивной секцией: пользователи, группы, занятия и посещаемость.

---

## Описание проекта

Проект разработан в рамках учебной практики.

Система позволяет:
- управлять пользователями (тренеры / участники)
- создавать группы
- назначать тренеров
- планировать занятия
- учитывать посещаемость

Цель - упростить организацию тренировочного процесса и контроль активности участников.

---

## Технологии

- Python
- Django
- SQLite
- HTML (Django Templates)

---

## Структура проекта
users/ # работа с пользователями
groups/ # управление группами
schedule/ # занятия
attendance/ # посещаемость

config/ # настройки проекта
templates/ # HTML-шаблоны

db.sqlite3 # база данных
manage.py # точка запуска

---

## Основные сущности

- **User** — пользователь (тренер / участник)
- **Group** — группа (с привязкой к тренеру)
- **Schedule** — занятие
- **Attendance** — посещаемость

Связи:
- один тренер → много групп (ForeignKey)
- участники ↔ группы (ManyToMany)
